### 2. 📈 添加数据统计面板

**改进内容**:
- ✅ 创建 StatsPanel 通用组件
- ✅ 所有分析页面显示关键统计指标
- ✅ 支持图标、颜色、趋势指示器
- ✅ 响应式网格布局
- ✅ 悬停动画效果

**新增文件**:
- `src/components/StatsPanel.vue` - 统计面板组件

**组件特性**:
```typescript
export interface StatItem {
  label: string           // 指标名称
  value: string | number  // 指标值
  icon: any              // 图标组件
  color: string          // 主题颜色
  trend?: {              // 可选趋势
    type: 'up' | 'down'
    value: string
  }
}
```

**统计指标示例**:
- **选品分析**: 数据行数、数据维度、数据质量、分析时间
- **热词分析**: 热词数量、分析平台、上升趋势、分析时间
- **带货分析**: 数据条数、数据类型、分析维度、分析时间

**用户价值**:
- 📊 关键指标一目了然
- 🎯 快速了解数据概况
- 📈 趋势变化直观展示
- 🎨 美观的视觉设计
