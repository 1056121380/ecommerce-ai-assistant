## 🏗️ 技术架构

### 前端技术栈
- **框架**: Vue 3 + TypeScript + Vite
- **UI 库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router
- **图表**: ECharts
- **Markdown**: marked + highlight.js
- **报告生成**: jsPDF + html2canvas + docx

### 后端技术栈
- **框架**: Tauri 2 (Rust)
- **HTTP 客户端**: reqwest
- **凭据存储**: Windows Credential Manager

### 开发工具
- **测试**: Vitest (41 个测试用例)
- **类型检查**: TypeScript (严格模式)
- **构建**: Vite + Rolldown
